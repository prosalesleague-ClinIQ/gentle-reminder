# AI Services (Phase 2)

Python FastAPI microservices for Gentle Reminder.

## Planned Services

- **Speech-to-Text** - Whisper integration for story transcription
- **Emotion Detection** - NLP emotional tone analysis
- **Memory Graph Builder** - Neo4j knowledge graph construction
- **Cognitive Decline Detector** - ML-based trend analysis
- **Digital Twin Model** - Predictive cognitive modeling

## Tech Stack

- Python 3.11+
- FastAPI
- PyTorch / Scikit-learn
- OpenAI Whisper
- Neo4j driver
