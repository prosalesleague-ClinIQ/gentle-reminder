# Vendor Vetting Worksheets

**Document ID:** VVW-001
**Version:** 1.0
**Effective Date:** 2026-04-20
**Purpose:** Pre-engagement due-diligence worksheets for outside vendors who will be paid success-fee, equity, retainer, or contingency. One worksheet per vendor; retain in the contact's GHL note.

---

## How to use

1. Copy the relevant section below into a new note (markdown / Google Doc / GHL note)
2. Fill in fields from public sources (website, LinkedIn, FINRA, USPTO, state bar, SEC EDGAR)
3. Flag any red item; stop the engagement and investigate before signing
4. Attach the completed worksheet to the contact record in GHL
5. Reference the completed worksheet ID in the engagement letter if executed

Worksheets:
- §1 Patent attorney (equity / contingency)
- §2 Placement agent / capital raiser (success fee)
- §3 Fractional CFO (retainer + potential equity)
- §4 Grant specialist (contingency / pay-upon-award)

## §2 Placement Agent / Capital Raiser Vetting Worksheet

**Critical:** paying success fees to an unregistered broker-dealer is **illegal under 15 U.S.C. § 78o** and can void the entire round. This worksheet is non-negotiable.

### Identity
- Firm name: __________
- Individual representative: __________
- Rep's Series 79 (or equivalent) license number: __________
- Firm CRD number: __________

### Verification (green / yellow / red) — do these FIRST
- [ ] FINRA broker-dealer registration verified: https://brokercheck.finra.org/firm/summary/[CRD] — status: ______ — date checked: ______
- [ ] Individual registration verified: https://brokercheck.finra.org/individual/summary/[CRD] — status: ______ — date checked: ______
- [ ] Series 79 (or relevant) license current: ______
- [ ] No FINRA disciplinary actions in last 10 years: ______
- [ ] No SEC enforcement actions: verified at https://www.sec.gov/litigation — date checked: ______
- [ ] No state securities regulator actions: ______

### If any FINRA / SEC red flag: STOP. Do not engage.

### Success fee structure
- Success fee %: ______ % of capital raised
- Fee tiers (if any): $______ at < $N raised, ______ % at ≥ $N
- Minimum fee: $______
- Retainer (if any): $______ — justified? ______
- Equity / warrant component: ______ % / ______ warrants
- Strike price: ______
- Vesting on equity: ______

### "Attributed investor" definition
This is where bad engagement letters hide landmines. Require explicit, narrow language.
- Definition in draft engagement letter: __________
- **Carveouts** for pre-existing investor relationships Founder had:
  - List of named pre-existing relationships (these investors do NOT trigger success fee): __________
- Named exclusions from attribution: __________
- Burden of proof: does Founder have to prove a pre-existing relationship (bad) or does agent have to prove attribution (good)? ______

### Tail period
- Length: ______ months (prefer ≤ 12 months)
- Coverage: only investors introduced by agent, or all closings in the period?
- Tail fee rate: ______ % (sometimes lower than active-period fee)

### Exclusivity
- [ ] Non-exclusive (preferred)
- [ ] Exclusive for ______ months (prefer ≤ 6 months if exclusive)
- Carve-outs for specific investors / categories: __________

### References (get 2-3 healthtech clients with completed raises)
- Reference 1: __________
- Reference 2: __________
- Reference 3: __________
- Date contacted: ______

### Red flags (STOP)
- [ ] Cannot verify FINRA / SEC
- [ ] Requests retainer without FINRA registration
- [ ] Proposes broad "attributed investor" definition with no carveouts
- [ ] Tail period > 18 months
- [ ] Exclusivity > 12 months
- [ ] Has been sanctioned by FINRA or SEC
- [ ] Won't name individual representative
- [ ] Requests signature before referrals complete

### Decision
- [ ] Proceed to engagement letter draft
- [ ] Reject — reason: ______
- [ ] Hold — next step: ______
