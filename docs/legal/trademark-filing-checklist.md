## Why file now

- **Protect the brand** before investor meetings generate public buzz that would alert competitors / trademark squatters.
- **Support fundraising narrative** — investors ask about trademark early; "filed" is a strong signal.
- **Block competitor adoption** — federal registration gives nationwide constructive use as of the filing date.
- **Enable ® symbol use** once registered (years 1-3 post-filing).

## What to file

### 1. Word mark — "GENTLE REMINDER"
- **Standard character mark** (not stylized) — broadest protection, covers any font/color/case.
- **Filing basis:** 1(b) **Intent to Use (ITU)** — we have clear intent but no marketplace use yet. Converts to 1(a) when you file a Statement of Use after launch.
- **TEAS form:** TEAS Plus ($250/class) — cheapest, requires using pre-approved ID of Goods/Services. TEAS Standard ($350/class) — more flexible, typically unnecessary for software.

### 2. Logo mark (optional — defer if logo isn't finalized)
- **Design mark** or **stylized mark** covering the logo + wordmark combination.
- File separately once the logo is final (design changes post-filing invalidate the registration).
- Can wait until post-seed.

## Pre-filing search (do this BEFORE filing)

1. **USPTO TESS search** — https://tmsearch.uspto.gov/
   - Exact search: `"GENTLE REMINDER"` in wordmark field
   - Phonetic / similar: `GENT*` AND `REMINDER*`
   - Check live + dead marks (dead marks can still block if they were recent and active again)
2. **Common-law search** — Google, LinkedIn, Twitter, App Store, Google Play for existing product use not registered
3. **Domain search** — Check if `gentlereminder.com` is held by a confusingly similar product
4. **Record findings** — Screenshots / PDFs of search results, saved in `/docs/legal/trademark-search-results/` before filing

**If search surfaces a confusingly similar registered mark in Class 9 or 44:** Do NOT file without counsel review. Risks: (a) refused application (lose $250-500), (b) cease-and-desist letter from mark holder.

## Who owns the mark — pre-entity vs post-entity

Two paths:

### Path A (recommended): File post-entity in the C-Corp's name
- Wait until the Delaware C-Corp is formed (this week).
- File directly under the entity. Entity is the registered owner from day one.
- **Pro:** Clean chain of title. No assignment needed later.
- **Con:** 1-week delay. Competitive risk during that week is low but non-zero.

### Path B (alternative): File now in founder's name + assign to entity post-formation
- File in Christo Mack's personal name immediately (today).
- Upon entity formation, execute a **trademark assignment** from founder to the C-Corp.
- Record assignment with USPTO (form TEAS Assignment Cover Sheet, $40 fee).
- **Pro:** Protects priority date immediately. Competitive block from day 0.
- **Con:** Extra paperwork, small assignment fee, slight risk of assignment gap if founder becomes unavailable.

**Decision:** Path A unless competitive threat emerges in the next 7 days. If a lookalike competitor launches or squatter files, switch to Path B.

## Filing mechanics (TEAS Plus, DIY or paralegal)

1. Create USPTO account at `https://uspto.gov`
2. Verify applicant identity (attorney login not required for pro se filing, but check current USPTO rules — foreign applicants require counsel; US applicants can self-file)
3. Complete TEAS Plus form:
   - Mark: GENTLE REMINDER (standard characters)
   - Applicant: [entity name] or "Christo Mack" depending on path
   - Classes: 9 + 44
   - Pre-approved Goods/Services ID (search "software" and "medical services" in USPTO's Trademark ID Manual)
   - Filing basis: 1(b) Intent to Use for both classes
   - Specimen: NOT required at ITU stage (required at Statement of Use)
4. Pay: $250 × 2 classes = $500
5. Receive serial number within 24 hours
6. Monitor USPTO TSDR (Trademark Status and Document Retrieval) weekly

## Post-filing timeline (expected)

| Stage | Timing | Action |
|-------|--------|--------|
| Filing receipt | Immediate | Save serial number |
| USPTO examining attorney assigned | ~3-4 months | Watch TSDR for Office Actions |
| First action (if any) | Month 4-6 | Respond within 6 months to Office Action |
| Publication in TM Gazette | Month 6-9 | 30-day opposition window |
| Opposition period ends | Month 7-10 | If no opposition, advances to Notice of Allowance |
| Notice of Allowance (ITU) | Month 8-12 | You have 36 months (with extensions) to file Statement of Use |
| Statement of Use + specimen | Post-launch | Must show mark in actual commercial use; $100/class fee |
| Registration issued | 2-8 weeks post-SOU | ® symbol authorized, 10-year renewable term begins |

**Budget for full lifecycle:** ~$650-$900 (filing + SOU + potential extensions) per Class for DIY. Add ~$1,500-$3,000 if using an attorney.

## Attorney vs DIY

**DIY recommended because:**
- Mark is distinctive (two generic English words combined in a non-obvious way, suggestive rather than descriptive for the product)
- No international filing needed yet (Madrid Protocol can happen Year 2+)
- USPTO TEAS Plus is designed for self-filers with pre-approved IDs
- Total cost DIY: $500 vs attorney-assisted $2,000-$5,000

**Get an attorney if:**
- TESS search surfaces similar marks
- You want international filing (Madrid Protocol / direct country filings)
- You plan Brand House architecture (multiple sub-marks)
- Office Action rejection requires substantive response

## Defensive measures until registered

- Use TM superscript `™` from day one (no registration required, signals claim to a mark)
- **Do NOT use ®** until registration issues — it's a false statement and a federal offense
- Monitor TESS quarterly for confusingly similar new filings
- Monitor app stores (App Store, Google Play) for lookalike apps
- Register `gentlereminder.com`, `gentlereminder.co`, `gentlereminder.app`, `gentlereminder.health` domains (if not already)
- Register Apple App Store / Google Play app name early
