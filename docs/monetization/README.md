## Critical Success Factors

1. **Don't miss the 12-month provisional deadline.** Priority lost = valuation crashes.
2. **Protect trade secrets alongside patents.** Keep numerical parameters confidential.
3. **Build the clinical evidence base early.** FDA 510(k) requires substantive data.
4. **Establish customer validation before Series A.** Pilots speak louder than projections.
5. **Maintain inventor assignment chain of custody.** Any gap can void patent rights.
6. **Pursue non-dilutive grants aggressively.** They stretch runway and validate scientific rigor.

## Escalation Protocol

If any of the following occurs, escalate to the board and patent attorney immediately:

- Competitor announces patent filing in overlapping space
- Key inventor departs without proper assignment
- FDA issues a change in SaMD classification guidance
- Public disclosure occurs before filing
- Prior art emerges blocking a Tier 1 claim
- Major strategic expresses acquisition interest
