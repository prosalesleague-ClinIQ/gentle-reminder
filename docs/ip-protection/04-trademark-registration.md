# 4. USPTO Trademark Registration

**Cost:** $250-$350 per class (TEAS Plus / TEAS Standard)
**Time:** 6-12 months to registration
**Legal effect:** Exclusive right to use the mark in commerce in your class(es); ability to sue for infringement; ® designation.

---

## Why Trademark Matters

Trademarks protect your **brand identity** — the name "Gentle Reminder" and any associated logos. They block competitors from launching similar-named products that confuse customers.

Without trademark registration:
- You have common-law rights only in your geographic area of use
- Competitors can use similar names in other regions
- Difficult to enforce against online/digital competitors (no geographic boundary)

## Pre-Filing Steps

### 1. Trademark Search (FREE)
Before filing, verify the name is available:
- **USPTO TESS** (free): https://tmsearch.uspto.gov/ (search "Gentle Reminder" in all classes)
- **State registrations**: https://www.uspto.gov/trademarks/search/state-trademark-information-links
- **Google search** for common-law use
- **Domain search**: Is gentlereminder.com (or .health) taken? What about similar marks?

**What to check:**
- Exact matches in Class 9 or 44
- Confusingly similar marks (e.g., "Gentle Reminders", "The Gentle Reminder")
- Pending applications that could block registration

If there's a conflicting prior mark, consider:
- Alternative brand name
- Coexistence agreement with existing holder (if mark is in different class)
- Strategic filing anyway (sometimes prior marks abandoned)

### 2. Specimen Preparation
USPTO requires proof that the mark is "used in commerce" (or intent to use). Acceptable specimens for Class 9 software:
- Screenshot of software showing the mark
- Screenshot of product web page
- User manual or packaging

For Class 44 medical services:
- Website showing services offered under the mark
- Brochure or advertisement

Keep specimens ready as PDFs or PNGs.

### 3. Date of First Use
Document when "Gentle Reminder" was first used in commerce (not just on your own computer — must have been used with a customer or publicly available).

- **First Use Anywhere** (in the US): [Date]
- **First Use in Commerce** (interstate): [Date]

If the mark is not yet in actual commercial use, file as **Intent-to-Use (ITU)** — you have 6 months (extendable to 36 months) to show actual use before the trademark is finalized.

## How to File

### Option A: TEAS Plus — $250 per class
**Most common for startups.** Uses USPTO-approved goods/services descriptions from the ID Manual. Strict formatting requirements.

### Option B: TEAS Standard — $350 per class
More flexibility in goods/services description. Use when TEAS Plus descriptions don't fit your product.

### Filing URL
https://teas.uspto.gov/initial/

### Required Information
1. Applicant name + address (Gentle Reminder, Inc. or founder name)
2. Mark (typed word or image file)
3. Goods/Services description (from USPTO ID Manual for TEAS Plus)
4. Filing basis (Use-in-Commerce or Intent-to-Use)
5. Specimen (if Use-in-Commerce)
6. First use dates (if Use-in-Commerce)
7. Declaration signed by AOR

### Class 9 Sample Description (TEAS Plus)
"Downloadable software for cognitive assessment of patients with dementia; Downloadable software for tracking cognitive health; Downloadable mobile applications for healthcare monitoring"

### Class 44 Sample Description (TEAS Plus)
"Providing online medical information regarding cognitive health; Providing a website featuring medical information in the field of dementia; Healthcare services, namely, cognitive assessment services"

## Post-Filing Process

### Phase 1: Examination (Months 3-6)
- USPTO examiner reviews application
- May issue "Office Action" requesting clarifications or raising refusals
- You have 3-6 months to respond to Office Actions (now 3 months under Trademark Modernization Act)

### Phase 2: Publication (Months 6-9)
- If accepted, mark is published in *Official Gazette*
- 30-day opposition period (third parties can oppose)
- If unopposed, proceeds to registration

### Phase 3: Registration (Months 9-12)
- Use-based applications: Registered ✓ (receive certificate)
- Intent-to-Use applications: Must file Statement of Use + specimen within 6 months

### Maintenance (Recurring)
- Between years 5-6 after registration: File Section 8 Declaration of Use ($225)
- Between years 9-10: File Section 8 + 9 Renewal ($425)
- Every 10 years thereafter: Renewal ($425)

---

## Using the ™ and ® Symbols

- **™ (Unregistered):** Use immediately — "Gentle Reminder™". No legal requirements.
- **® (Registered):** Use ONLY after registration is final. Using ® before registration can be found fraudulent.

Apply ® in all marketing materials, product screens, and packaging post-registration.

## International Trademark Strategy (Future)

Once US registration is filed, you have 6 months to claim priority internationally:
- **Madrid Protocol** (WIPO): file one application covering 130+ countries
  - Base fee: CHF 653 + per-country fee
  - Typical cost for 5 priority countries: $3,000-$5,000
- **Direct national filings** in key markets (EU via EUIPO, UK, Canada, Australia, Japan)

## Common Mistakes to Avoid

1. **Filing before search** — discovering a conflicting mark after filing wastes money
2. **Filing TEAS Plus with non-ID-manual descriptions** — rejected, have to refile or convert to TEAS Standard
3. **Using ® before registration** — potentially fraudulent
4. **Not using the mark for 3 consecutive years** — mark can be cancelled for non-use
5. **Not filing Section 8** between years 5-6 — mark cancelled

---

## Self-Filing vs. Attorney

- **Self-filing** is legally allowed and saves attorney fees
- **Attorney-assisted filing**: $500-$2,000 in addition to USPTO fees
- For simple word marks in standard classes, self-filing is fine
- For complex cases (logos, multiple classes, international strategy), use a trademark attorney

**Recommended approach:** Self-file the primary "Gentle Reminder" word mark now; engage trademark attorney for logo, international, and product mark filings after seed close.
