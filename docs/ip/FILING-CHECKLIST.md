# USPTO Provisional Patent Filing Checklist

**CONFIDENTIAL — Attorney-Client Privileged**

This checklist walks through filing a USPTO provisional patent application via EFS-Web (the Electronic Filing System). Follow this checklist for each of the 23 Gentle Reminder IPs.

## Phase 0: Prerequisites (Before Filing Any Patents)

### 0.1 Legal Entity Setup
- [ ] Incorporate company (recommend Delaware C-Corp for future fundraising)
- [ ] Obtain EIN from IRS
- [ ] Open corporate bank account for USPTO fee payments
- [ ] Register for USPTO.gov account (free)
- [ ] Register for EFS-Web filing account (free, requires USPTO.gov account)
- [ ] Register for Pay.gov account (links to corporate bank or credit card)

### 0.2 Engage Patent Attorney
- [ ] Interview 3 patent firms — recommended candidates:
  - **Boutique software/medical device specialists:** Carr & Ferrell, Shay Glenn LLP, Medlen & Carroll LLP
  - **Full-service firms:** Cooley LLP, Wilson Sonsini Goodrich & Rosati, Fenwick & West
  - **Lower-cost options:** LegalForce Trademarks, Rocket Lawyer (for basic filings only — NOT recommended for strategic patents)
- [ ] Execute attorney engagement letter
- [ ] Establish billing arrangements (hourly vs fixed fee — recommend fixed fee for provisionals at $500-$1,500 each)

### 0.3 Inventor Setup
- [ ] Identify all named inventors for each IP (every person who made an inventive contribution)
- [ ] Collect: full legal name, residential address, citizenship, email, phone
- [ ] Have each inventor sign an Invention Assignment Agreement (IAA) transferring rights to the company — this is typically in the employment agreement, but a separate confirming assignment for each IP is recommended
- [ ] Document invention dates (conception + reduction to practice) for each IP

### 0.4 Entity Status Determination
Choose one:
- [ ] **Micro-entity** ($300 fee): Annual gross income below $223,740 AND no inventor has been named on more than 4 prior filed applications AND applicant has NOT assigned rights to an entity ineligible for micro-entity status
- [ ] **Small-entity** ($600 fee): Fewer than 500 employees AND not assigned to a large entity
- [ ] **Large-entity** ($1,500 fee): Default if no special status claimed

File USPTO Form SB/15A (Micro-Entity Status) or SB/15B (Certification of Micro-Entity Status — Gross Income Basis) if claiming micro-entity.

## Phase 1: Draft Preparation (Per IP)

### 1.1 Complete the Provisional Draft
- [ ] Fill out all sections of the template ([TEMPLATE-provisional-patent.md](TEMPLATE-provisional-patent.md))
- [ ] Ensure Section 7 (Detailed Description) contains enabling disclosure
- [ ] Include at least 3 independent claims and 5 dependent claims
- [ ] Write 150-word abstract (per 37 CFR 1.72(b))
- [ ] Compile drawings (if any) as separate PDF files

### 1.2 Prior Art Search
- [ ] Search USPTO PAIR for keyword matches
- [ ] Search Google Patents
- [ ] Search IEEE Xplore, PubMed for academic prior art
- [ ] Document results in [PRIOR-ART-SEARCH.md](PRIOR-ART-SEARCH.md)
- [ ] Optional: Professional prior art search ($1,000-$3,000 per IP) — recommended for Tier 1 IPs

### 1.3 Attorney Review
- [ ] Submit draft to patent attorney
- [ ] Attorney performs formal review (typically 2-4 hours per draft at $300-$500/hour)
- [ ] Revise based on attorney feedback
- [ ] Attorney signs off on final draft

## Phase 2: Filing (Per IP)

### 2.1 Prepare Filing Documents
Required for each provisional application:
- [ ] **Specification** (the draft document) as PDF
- [ ] **Drawings** (if any) as separate PDF
- [ ] **USPTO Form SB/16** — Provisional Application for Patent Cover Sheet (downloadable from uspto.gov)
  - Title
  - Applicant information
  - Inventor names and addresses
  - Correspondence address
  - Entity status (micro/small/large)
- [ ] **Application Data Sheet (ADS)** — USPTO Form AIA/14 (strongly recommended)
- [ ] **Fee Transmittal** — auto-calculated by EFS-Web

### 2.2 File Via EFS-Web
1. [ ] Log in to EFS-Web: https://efs.uspto.gov/efile/portal/efs-registered
2. [ ] Select "Register/Log In" → "New Application" → "Provisional Application under 35 USC 111(b)"
3. [ ] Upload specification PDF (single file — do not split into sections)
4. [ ] Upload drawings PDF (if any)
5. [ ] Upload completed SB/16 cover sheet
6. [ ] Upload ADS (optional but recommended)
7. [ ] Claim entity status (micro/small/large)
8. [ ] Review filing summary
9. [ ] Pay via Pay.gov (EFT, credit card, or USPTO Deposit Account)
10. [ ] Submit and save the filing receipt (contains Application Number — critical for tracking)

### 2.3 Post-Filing Actions
- [ ] Save Filing Receipt PDF to secure storage
- [ ] Record Application Number and Filing Date in internal IP tracker
- [ ] Set calendar reminder for **11 months post-filing** (conversion deadline warning)
- [ ] Set calendar reminder for **12 months post-filing** (conversion deadline — HARD deadline)

## Phase 3: Post-Filing Management

### 3.1 Within 2 Months of Filing
- [ ] Receive USPTO Filing Receipt via postal mail (or Patent Center)
- [ ] Verify all inventor names, addresses, and application data are correct
- [ ] Request corrections via Request for Correction of Filing Receipt if needed (USPTO Form PTO/SB/41)

### 3.2 Months 3-6 Post-Filing
- [ ] Professional prior art search (if not done pre-filing)
- [ ] Begin non-provisional drafting for highest-priority IPs
- [ ] Evaluate international filing strategy (PCT vs direct national phase)

### 3.3 Months 9-11 Post-Filing
- [ ] Final prior art evaluation
- [ ] Decision: Convert to non-provisional? Abandon? File continuation?
- [ ] If converting: Begin formal non-provisional drafting (8-12 weeks lead time)
- [ ] If filing PCT: Prepare PCT application (must file within 12 months to claim priority)

### 3.4 Month 12 (HARD DEADLINE)
- [ ] **File non-provisional application (claiming priority to provisional)** — missing this destroys priority date
- [ ] **OR file PCT application claiming priority to provisional**
- [ ] **OR formally abandon** (provisional automatically expires at 12 months)

Typical cost for non-provisional filing:
- Attorney fees: $8,000-$15,000 per application (patent drafting + prosecution)
- USPTO filing fees: $800-$1,600 (micro/small/large entity)
- USPTO examination fees: $250-$1,000
- Prior art search: $600-$800
- **Total: $10,000-$20,000 per non-provisional conversion**

## Phase 4: Prosecution (After Non-Provisional Filing)

### 4.1 Office Actions
- Expect 1-3 Office Actions from USPTO examiner (typically 18-36 months after filing)
- Response cost: $3,000-$8,000 per Office Action
- Total prosecution cost: $15,000-$40,000 per application over 3-5 years

### 4.2 Grant
- Notice of Allowance → pay issue fee ($250-$1,000)
- Patent issues ~3 months after Notice of Allowance
- Maintenance fees due at 3.5, 7.5, and 11.5 years: $800, $1,800, $3,700 respectively (micro-entity)

## Common Mistakes to Avoid

1. **Public disclosure before filing** — Any public disclosure (blog post, demo, published paper, pitch deck) before filing starts a 12-month clock in the US and destroys foreign patent rights in most countries. **File before public disclosure.**

2. **Insufficient enabling disclosure** — Provisional must describe the invention in enough detail that a Person of Ordinary Skill in the Art (POSITA) could reproduce it. Vague descriptions fail.

3. **Missing the 12-month deadline** — Priority date is lost forever if non-provisional is not filed within 12 months.

4. **Incorrect inventor naming** — Only actual inventive contributors should be named. Adding non-inventors (including company executives or funders) can invalidate the patent.

5. **Not filing assignment** — Inventors own the patent by default. Ensure invention assignment agreement is signed and recorded with USPTO.

6. **Using provisional as a placeholder** — Provisionals must contain the substantive disclosure. A "cover page only" provisional provides no priority protection for subject matter not disclosed.

7. **Missing micro-entity certification** — If claiming micro-entity status, must file SB/15A. Otherwise USPTO charges small-entity or large-entity rates.

## Emergency Contact Protocol

If facing an imminent public disclosure (conference talk, published paper, investor deck release) with an unfiled invention:
1. File a "rush provisional" — attorney can draft and file within 1-2 business days
2. Cost: $2,500-$5,000 for rush attorney fees + $300 USPTO filing fee
3. The rush provisional must still contain enabling disclosure; do not cut corners on substance
