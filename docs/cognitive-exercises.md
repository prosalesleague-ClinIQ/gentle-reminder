# Cognitive Exercise Design

## Exercise Types

### Orientation Exercises

Based on reality orientation therapy. Reinforce awareness of:

| Exercise | Prompt | Domain |
|----------|--------|--------|
| Date | "What day of the week is it?" | Orientation |
| Name | "What is your name?" | Orientation |
| Location | "What city are we in?" | Orientation |
| Month | "What month are we in?" | Orientation |

### Identity Exercises

Based on reminiscence therapy. Reinforce family relationships:

| Exercise | Prompt | Domain |
|----------|--------|--------|
| Recognition | Show photo, "Who is this?" | Identity |
| Relationship | "How is [name] related to you?" | Identity |

### Memory Exercises

Cognitive stimulation through recall:

| Exercise | Prompt | Domain |
|----------|--------|--------|
| Category | "Name three fruits/animals/colors" | Memory |
| Object | "What do you use to [action]?" | Memory |
| Sequence | "Remember these words: [list]" | Memory |

## Scoring

### Gentle Scorer Rules

1. Every answer produces one of three feedback types (NEVER negative):
   - **Celebrated** (score 1.0) - Correct
   - **Guided** (score 0.3-0.5) - Partially correct
   - **Supported** (score 0.0) - Incorrect, answer provided warmly

2. Partial matching is generous:
   - "Port" for "Portland" = Guided (partial match)
   - Preferred name counts ("Maggie" for "Margaret")
   - Case insensitive

3. Category exercises count valid items:
   - 3/3 = Celebrated
   - 1-2/3 = Guided
   - 0/3 = Supported

## Session Structure

Each session follows a fixed order to reduce confusion:

1. Orientation (2-3 questions)
2. Identity (1-2 questions)
3. Memory (1-2 questions)

Total: 5-7 exercises, 5-7 minutes

## Fatigue Detection

Response time monitoring (not shown to patient):
- If last 3 exercises take 50%+ longer than first 3 = fatigue signal
- Logged for caregiver review
