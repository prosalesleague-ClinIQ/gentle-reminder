## TIPS FOR STRONG BIOSKETCH

1. **Lead with relevance to dementia assessment.** Reviewers skim; the first line must establish your fit.
2. **Use measurable outcomes.** "Led development of X that achieved Y" beats vague experience.
3. **Patent filings count for SBIR applicants** even if not yet issued — include them under Contributions.
4. **If you lack peer-reviewed publications**, focus on: deployed software with users, regulatory submissions, technical reports, conference presentations, issued patents.
5. **The eRA Commons ID is required** — register well before submission (era.nih.gov).
