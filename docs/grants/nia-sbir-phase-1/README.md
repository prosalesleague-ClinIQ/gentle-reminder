## Key Review Criteria (NIH SBIR Phase I)

NIH reviewers score on a 1-9 scale (1 = exceptional, 9 = poor):
1. **Significance** — Clinical need + market opportunity
2. **Innovation** — Novel approach vs. state of the art
3. **Approach** — Scientific rigor of the proposed Phase I work
4. **Investigator(s)** — PI + team qualifications
5. **Environment** — Institutional resources
6. **Commercialization Potential** — Path to market

**Target score:** 10-30 impact score is fundable. Our drafts target ≤25.
